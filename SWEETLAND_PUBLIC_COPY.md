# Sweet Land Public Copy

YouTube概要欄、Webサイト、配信中の固定コメントに使える文言集。

## YouTube Description

```text
Sweet Land配信へようこそ。

この配信では、視聴者のみなさんが順番待ちに参加して、オンラインでSweet Landを操作できます。

参加の流れ:
1. 配信画面で受付開始を待つ
2. Webページでレーンを選んで順番待ちに参加
3. 自分の番になったら、制限時間内に操作

注意事項:
- 一人あたりのプレイ回数には上限があります。
- 通信やカメラ映像には遅延が発生する場合があります。
- 景品詰まりや安全確認のため、一時停止する場合があります。
- 不正操作、連打、荒らし行為があった場合は参加を制限します。
- 運営判断により、順番やプレイを調整する場合があります。

トラブル時は配信内で案内します。
楽しく安全に遊びましょう。
```

## How To Play

```text
1. 受付開始を待つ
配信中に「受付開始」と表示されたら参加できます。

2. レーンを選んで並ぶ
好きなレーンを選び、表示名を入力して順番待ちに入ります。

3. 自分の番で操作する
左右ボタンで位置を調整し、タイミングを見てすくいます。
```
