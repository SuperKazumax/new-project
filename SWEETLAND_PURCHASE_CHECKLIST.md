# Sweet Land 4 Purchase Checklist

## Purpose

Sweet Land 4を購入する前に、搬入、電源、状態、遠隔操作化のしやすさを確認する。

## Must Confirm Before Buying

| Result | Check | Notes |
| --- | --- | --- |
|  | 本体がSweet Land 4である |  |
|  | 動作確認済みである |  |
|  | 実際に動いている動画がある |  |
|  | 電源仕様が分かる |  |
|  | サイズが分かる |  |
|  | 重さが分かる |  |
|  | 搬入経路に入る |  |
|  | 設置場所に置ける |  |
|  | 鍵がある |  |
|  | 説明書または配線図がある |  |
|  | ボタンが正常に効く |  |
|  | シャベル動作が正常 |  |
|  | 回転台が正常 |  |
|  | 景品落とし口が正常 |  |
|  | コイン/クレジット設定が確認できる |  |
|  | 異音や焦げ臭さがない |  |

## Seller Questions

購入前に出品者へ聞く。

```text
1. 電源を入れて通常プレイできますか？
2. 左右移動、すくう操作、回転台は正常ですか？
3. 鍵はありますか？
4. 説明書や配線図はありますか？
5. 電源仕様、サイズ、重さを教えてください。
6. 故障、修理歴、不具合はありますか？
7. 搬出時に分解が必要ですか？
8. 配送方法はどうなりますか？
9. 実際に動いている動画を見せてもらえますか？
10. ボタン部分の写真を見せてもらえますか？
```

## Remote Control Suitability

遠隔操作化しやすいか確認する。

| Result | Check | Notes |
| --- | --- | --- |
|  | 操作ボタンが外から押しやすい位置にある |  |
|  | ボタン周辺にサーボ固定スペースがある |  |
|  | カメラを置ける位置がある |  |
|  | 盤面が照明で見やすい |  |
|  | シャベル位置がカメラで追える |  |
|  | 手動緊急停止できる方法がある |  |
|  | 通常プレイ時の1ターン時間が測れる |  |
|  | 長押しが必要か短押しか分かる |  |

## Red Flags

次に当てはまる場合は慎重にする。

- 動作未確認。
- 鍵なし。
- 電源仕様が不明。
- 搬入サイズが不明。
- 出品者が動画を出せない。
- ボタンやシャベルに不具合がある。
- 異音、焦げ臭さ、漏電の疑いがある。
- 配線が大きく改造されている。
- 返品不可で状態説明が少ない。

## Minimum Buying Condition

最初の1台は、次を満たすものを優先する。

- 通常プレイできる。
- 鍵がある。
- 電源仕様が分かる。
- 動作動画がある。
- 操作ボタンが正常。
- カメラを置けそう。
- 搬入できる。

## After Arrival

届いたらすぐにやる。

- [ ] 外装を写真で記録する。
- [ ] 配線を触る前に内部を写真で記録する。
- [ ] 電源投入前に破損や異物を確認する。
- [ ] 電源投入を短時間だけ行う。
- [ ] 通常プレイを1回だけ試す。
- [ ] 操作ボタンの反応を確認する。
- [ ] カメラ位置の候補を決める。
- [ ] 実機改造前の状態を動画で残す。
