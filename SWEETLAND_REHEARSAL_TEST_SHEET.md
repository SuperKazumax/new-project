## 2. Stream Phase Flow

| Result | Step | Expected | Notes |
| --- | --- | --- | --- |
|  | `21:00 開始` を押す | 配信開始状態になる |  |
|  | `受付開始` を押す | 新規順番待ちが可能になる |  |
|  | `ラスト募集` を押す | 締め切り前メッセージが出る |  |
|  | `終了準備` を押す | 新規順番待ちが止まる |  |
|  | `配信終了` を押す | 順番待ちと操作が止まる |  |
|  | `配信前に戻す` を押す | 待機状態に戻る |  |

## 3. Queue Tests

| Result | Step | Expected | Notes |
| --- | --- | --- | --- |
|  | 受付前に参加を押す | 参加できない |  |
|  | 受付中に1人目が参加する | キューに入る |  |
|  | 受付中に2人目が同じレーンへ参加する | 2番目に入る |  |
|  | 2人目が別レーンへ参加する | レーン状態が混ざらない |  |
|  | 参加者が順番待ちから抜ける | キューから消える |  |
|  | 同じユーザーが複数回参加する | 重複参加できない |  |
|  | 終了準備中に新規参加する | 参加できない |  |

## 4. Play Tests

| Result | Step | Expected | Notes |
| --- | --- | --- | --- |
|  | `次の人を開始` を押す | 先頭ユーザーに操作権が渡る |  |
|  | 操作権なしユーザーが左右操作する | 操作できない |  |
|  | 操作権ありユーザーが左へ動かす | 実機または画面上で左へ動く |  |
|  | 操作権ありユーザーが右へ動かす | 実機または画面上で右へ動く |  |
|  | 操作権ありユーザーがすくう | 1回分を消費してターン終了 |  |
|  | ターン時間切れまで待つ | 自動でターン終了 |  |
|  | 残り回数0で操作する | 操作できない |  |
