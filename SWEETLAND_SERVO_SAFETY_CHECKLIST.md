# Sweet Land Servo Safety Checklist

## Purpose

外付けサーボやソレノイドでSweet Land 4の物理ボタンを押す前に、安全確認を行う。

このチェックリストは、実機内部の配線を触らない外付け方式を前提にする。

## Safety Principle

最初の実機操作は、次の条件を満たすまで行わない。

- すぐ止められる。
- 実機を改造していない。
- サーボ電源を切れば物理的に止まる。
- 人が実機の近くで確認している。
- カメラで状態が見える。
- 連打や長押しが制限されている。

## 4. First Attached Test

実機につけた最初のテスト。

条件:

- 実機近くに人がいる。
- YouTube公開配信ではない。
- Webアプリは `safe` またはテスト専用状態。
- いつでもサーボ電源を切れる。

| Result | Step | Expected | Notes |
| --- | --- | --- | --- |
|  | サーボを固定する | ボタンに軽く当たる |  |
|  | 電源OFFで手動操作する | 手動操作できる |  |
|  | サーボ電源ON | 勝手に押さない |  |
|  | 1回だけ短く押す | ボタンが反応する |  |
|  | すぐ戻る | 押しっぱなしにならない |  |
|  | 緊急停止する | サーボ出力が止まる |  |
|  | 固定位置を再確認 | ずれていない |  |

## 8. Stop Conditions

次の場合は即停止する。

- サーボが押しっぱなしになる。
- 固定具がずれる。
- ボタン以外に干渉する。
- 実機から異音がする。
- 焦げ臭い。
- カメラ映像が見えない。
- 緊急停止が効かない。
- 管理者が状態を把握できない。

## 9. Go / No-Go

公開テスト前の判定。

- [ ] GO: サーボ実機テストを継続できる
- [ ] HOLD: 固定や制御を直して再テスト
- [ ] NO-GO: 実機操作を中止

## 10. Required Before Public Use

公開配信で使う前に必須。

- [ ] 物理緊急停止がある。
- [ ] Web緊急停止がある。
- [ ] safe / armed / emergency modeが動く。
- [ ] 押しっぱなしにならない。
- [ ] 連打制限がある。
- [ ] 操作ログが残る。
- [ ] 実機近くに確認者がいる。
- [ ] 30分以上の身内テストを通した。
