# Sweet Land Hardware Shopping List

## Purpose

Sweet Land 4を遠隔操作配信するために必要になりそうな機材を整理する。  
具体的な製品名は、実機のサイズや置き場所が決まってから選ぶ。

## Priority A: Streaming Basics

まず配信として成立させるために必要。

| Item | Purpose | Notes |
| --- | --- | --- |
| 盤面カメラ | シャベル、景品、落とし口を映す | 低遅延が重要 |
| 全体確認カメラ | 実機全体やトラブル確認 | 管理者用でもよい |
| 照明 | 盤面を明るくする | 反射に注意 |
| カメラ固定具 | 映像を安定させる | 三脚、アーム、クランプ |
| 有線LAN | 配信と操作の安定化 | Wi-Fiだけにしない |
| 配信用PC | YouTube配信と管理 | 既存PCでまず試す |
| 電源タップ | 実機、PC、カメラ、照明 | 容量確認 |

## Priority B: Safe External Control

実機を改造せず、外からボタンを押すために必要。

| Item | Purpose | Notes |
| --- | --- | --- |
| サーボまたはソレノイド | 物理ボタンを押す | まずは外付け方式 |
| 固定ブラケット | 押す位置を固定 | ずれないことが大事 |
| 小型制御PC | サーボ制御 | Raspberry Pi等 |
| サーボ電源 | 操作機構の電源 | 実機電源と分ける |
| 物理緊急停止スイッチ | 操作機構の電源を切る | 必須 |
| 配線保護材 | ケーブル保護 | 引っ掛かり防止 |

## Priority C: Electrical Interface

内部配線に接続する場合。後回し。

| Item | Purpose | Notes |
| --- | --- | --- |
| テスター | 電圧確認 | 必須 |
| 絶縁I/O | 実機と制御側を分離 | 安全のため |
| リレーモジュール | ボタン入力の代替 | 仕様確認後 |
| フォトカプラ | 信号絶縁 | 仕様確認後 |
| ヒューズ | 保護 | 電気担当者と確認 |
| 端子台 | 配線整理 | 復旧しやすくする |

## Priority D: Operations

運用を楽にするもの。

| Item | Purpose | Notes |
| --- | --- | --- |
| 管理者用モニター | キュー、ログ、カメラ確認 | 配信画面とは別 |
| UPS | 瞬断対策 | 予算次第 |
| 温度計 | 実機や電源周りの確認 | 長時間配信用 |
| ラベル | 配線やボタンの識別 | トラブル時に効く |
| 記録用ノート | 改善メモ | デジタルでも可 |

## Do Not Buy Yet

実機確認前に買うと無駄になる可能性が高いもの。

- 内部配線用のリレー一式
- 専用基板
- 高価な産業用カメラ
- 大量のサーボ
- 実機サイズに依存する固定具

## First Safe Prototype

最初に目指す構成。

```text
Web app
↓
Local server
↓
Raspberry Pi or small PC
↓
Servo controller
↓
External servo presses the original button
```

この方式なら、実機の内部配線を触らずに遠隔操作の検証ができる。
